# JVM Landing Page

Este projeto é uma landing page desenvolvida em React com TypeScript. O objetivo é apresentar uma seção de serviços com um design moderno e responsivo.

## Estrutura do Projeto

O projeto possui a seguinte estrutura de diretórios:

```
jvm-landing-page
├── src
│   ├── components
│   │   └── Service-Section
│   │       ├── index.tsx          # Componente responsável por renderizar a seção de serviços
│   │       └── CircleBackground.tsx # Componente que renderiza um círculo com fundo
│   └── types
│       └── index.ts               # Tipos e interfaces utilizados em todo o projeto
├── package.json                    # Configuração do npm com dependências e scripts
├── tsconfig.json                   # Configuração do TypeScript
└── README.md                       # Documentação do projeto
```

## Instalação

Para instalar as dependências do projeto, execute o seguinte comando:

```
npm install
```

## Execução

Para iniciar o projeto em modo de desenvolvimento, utilize:

```
npm start
```
